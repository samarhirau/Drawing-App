Basic Drawing App
A simple and intuitive drawing application built with HTML, CSS, and JavaScript. This lightweight app allows users to draw on a canvas using their mouse or touch input on mobile devices. It's perfect for quick sketches, doodling, or just having some creative fun.

Features:
Pen Tool: Freely draw with a customizable pen size and color.
Eraser Tool: Easily erase parts of your drawing.
Color Picker: Choose from a wide range of colors for your pen.
Clear Canvas: Start fresh with a blank canvas at any time.
Save Drawing: Save your masterpiece as an image file to your device.


How to Use:
Open the app in your web browser.
Select the pen tool to start drawing. Choose your desired pen color and size.
Use the eraser tool if you need to remove any part of your drawing.
Click the "Save" button to download your drawing as an image file.
To start a new drawing, click the "Clear" button to clear the canvas.


Technologies:
HTML5: For structuring the web app's content.
CSS3: For styling and making the app responsive.
JavaScript: For implementing the app's functionality and interactivity.
